# ClassmateHub 同学录管理系统

**版本号：V0.1**

> ⚠️ 注意：本项目目前尚未进入正式开发阶段，正处于框架搭建过程中。

## 项目概述

ClassmateHub 是一个现代化的全栈同学录管理系统，采用前后端分离架构，旨在帮助用户轻松维护同学信息，促进校友之间的联系与互动。系统提供了用户认证、班级管理、同学信息管理、信息隐私控制等核心功能，为用户打造一个安全、便捷的同学录管理平台。

## 项目目标

- 建立一个集中化的同学信息管理平台
- 方便班级管理员维护班级信息和成员
- 保护用户隐私，提供灵活的信息可见性控制
- 促进同学之间的联系与互动
- 提供简洁易用的用户界面和操作流程

## 核心功能

### 1. 用户管理
- 用户注册与登录
- 个人信息管理
- 密码修改与重置
- 基于角色的权限控制（管理员/普通用户）

### 2. 班级管理
- 创建、编辑、删除班级
- 查看班级列表和详情
- 管理班级成员（添加/移除）
- 班级信息维护

### 3. 同学信息管理
- 查看同学列表
- 查看同学详细信息
- 更新个人信息
- 字段级信息可见性控制（公开、仅同学可见、私人）

### 4. 系统特性
- 安全的用户认证机制（JWT）
- 统一的信息管理界面
- 灵活的隐私控制
- 响应式设计，支持多设备访问
- 前后端分离架构

## 快速开始

### 环境要求
- Java 17 或更高版本
- Node.js 16 或更高版本
- MySQL 8.0 或更高版本
- Maven 3.6 或更高版本

### 后端启动

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd ClassmateHub
   ```

2. **配置数据库**
   - 创建名为 `classmatehub` 的数据库
   - 修改后端配置文件 `backend/src/main/resources/application.yml` 中的数据库连接信息

3. **启动后端服务**
   ```bash
   cd backend
   mvn spring-boot:run
   ```

### 前端启动

1. **安装依赖**
   ```bash
   cd frontend
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问系统**
   - 前端默认运行在 `http://localhost:3000`
   - 后端API服务运行在 `http://localhost:8080`
   - Vite代理已配置将 `/api` 请求转发到后端
