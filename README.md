## 项目概述

ClassmateHub 是一个现代化的全栈同学录系统，采用前后端分离架构，旨在帮助用户轻松维护同学信息，促进校友之间的联系与互动。系统提供了用户认证、班级管理、同学信息管理、信息隐私控制、访客查看等核心功能，为用户打造一个安全、便捷的同学录平台。

## 项目目标

- 建立一个集中化的同学平台
- 方便管理员维护班级信息和成员
- 方便同学查看班级其他同学信息
- 保护用户隐私，提供灵活的信息可见性控制
- 促进同学之间的联系与互动
- 提供简洁易用的用户界面和操作流程

## 技术栈

### 后端技术栈
| 类别 | 技术 | 版本 | 用途 |
|------|------|------|------|
| 基础语言 | Java | 17+ | 开发语言 |
| 框架 | Spring Boot | 3.5.10 | 应用框架 |
| 持久层 | MyBatis | 3.5.16 | ORM框架 |
| 数据库 | MySQL | 8.x | 数据存储 |
| *认证* | *JWT* | *0.12.5* | *无状态认证* |
| 构建工具 | Maven | 3.9.6 | 依赖管理 |
| 代码简化 | Lombok | 1.18.30 | 减少样板代码 |
| *安全* | *Spring Security* | *6.3.1* | *密码加密* |
| JSON处理 | Jackson | 2.17.2 | JSON序列化/反序列化 |

### 前端技术栈
| 类别        | 技术         | 简要说明                        |
| ----------- | ------------ | ------------------------------- |
| 前端框架    | Vue.js 3     | 渐进式 JavaScript 框架          |
| UI 组件库   | Element Plus | 为 Vue 3 设计的桌面端组件库     |
| 路由管理    | Vue Router   | Vue.js 官方路由管理器           |
| 构建工具    | Vite         | 下一代前端构建工具              |
| 类型检查    | TypeScript   | JavaScript 的超集，提供类型安全 |
| HTTP 客户端 | Axios        | 基于 Promise 的 HTTP 客户端     |

### 1. 用户管理
- 用户注册与登录
- 个人信息管理
- 密码修改与重置
- 基于角色的权限控制（管理员/普通用户）

### 2. 班级管理
- 创建、编辑、删除班级
- 查看班级列表和详情
- 管理班级成员（添加/移除）
- 班级信息维护

### 3. 同学信息管理
- 查看同学列表
- 查看同学详细信息
- 更新个人信息

### 4. 系统特性
- 安全的用户认证机制（JWT）
- 统一的信息管理界面
- 灵活的隐私控制
- 响应式设计，支持多设备访问
- 前后端分离架构
